MinecraftProfilePro

MCPP is an open source Minecraft (Bukkit and Spigot) plugin and PHP script that allows server owners and players to know what their current stats are in game. For example: blocks mined, players killed, mobs killed etc.

Welcome to the MCPP Home Page.

On this page, you'll find a small guide that tells you how to install MCPP on your Minecraft (BUKKIT/SPIGOT) server.

If you're using a plain linux distro, you've got this fairly easy!

Step 1) Install some sort of web server... we won't go through this, there are tonnes of tutorials

Step 2) Download MCPP as a tarball and extract

$ wget https://github.com/matrixdevuk/MinecraftProfilePro/tarball/master
$ tar xvf <name of downloaded archive here>

Step 3) Move things

$ mv plugin_build/* /path/to/your/minecraft/server/plugins/folder
$ mv web/* /path/to/your/web/server

Step 4) Edit the configurations for both the plugin and web

$ vi /path/to/web/configuration/mysql/main.json
$ vi /path/to/web/configuration/web/main.json
$ vi /path/to/web/configuration/minecraft/main.json
$ vi /path/to/minecraft/plugins/mcpp/*

Step 5) Visit YOURWEBSITE.co/install.php

Step 6) Reload your server

Once you've done all the above on linux, you're done! Just /reload your server and watch as the stats start to count!

Note: we update the stats every 5 or so minutes, so there will be a little wait, but it's to reduce stress on your MySQL server!
